This report gives Cycle length and frequency information for sensors that measure whether a device is On or Off. Normally such a sensor has the value of 1 when the device is On and 0 when it is off. If the Sensor selected for this report instead has continuous values, this report determines a threshold midway between the minimum and maximum values of the sensor, and classifies readings above that threshold as representing On and those below as representing Off.
